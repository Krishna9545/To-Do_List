✅ To-Do List Web App : 
A simple and interactive To-Do List Application built using HTML, CSS, and JavaScript.
This app helps users manage daily tasks efficiently with features like adding, deleting, completing, and storing tasks.

🚀 Features : 
➕ Add new tasks
✅ Mark tasks as completed
❌ Delete individual tasks
🧹 Clear all completed tasks
📊 Live count of remaining tasks
💾 Data stored in localStorage (tasks saved even after refresh)
🎨 Clean and responsive UI


🛠️ Technologies Used : 
HTML5 – Structure of the app
CSS3 – Styling and responsive design
JavaScript (Vanilla JS) – Logic and functionality
LocalStorage API – Data persistence


📂 Project Structure : 
📁 Todo List Project
│── index.html     # Main HTML file
│── styles.css     # Styling file
│── app.js         # JavaScript logic


💡 How It Works : 
1) User enters a task and clicks Add.
2) Task is added to the list and saved in localStorage.


User can:
✔️ Mark task as completed
❌ Delete task
🧹 Clear all completed tasks
Remaining tasks count updates automatically.


▶️ How to Run : 
1) Clone the repository:
    git clone https://github.com/your-username/todo-list-app.git
2) Open index.html in your browser.
